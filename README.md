# Tree Viewer Application

## Requirements

- **Node.js**: v20.19.0, v22.12.0 or higher (recommended: v22.20.0 LTS)
- **npm**: 8.0.0 or higher


## Quick Start

1. **Install all dependencies:**

   ```bash
   npm run install:all
   ```

2. **Run both frontend and backend:**
   ```bash
   npm run dev
   ```

## Endpoints

- **Frontend**: `http://localhost:4200`
- **Backend API**: `http://localhost:3000`
